# **App Name**: Lapides Eye Care

## Core Features:

- Homepage Structure: Display a landing page with sections for Hero, About the Doctor, Services, Clinic Schedule, and Contact.
- Contact Form: Implement a contact form with fields for Full Name, Email, Mobile, and Message that sends directly to the doctor's email address.
- Doctor Bio Section: Present comprehensive information about the doctor's background, expertise, and approach to patient care.
- Service Showcase: Showcase the services offered by Dr. Lapides including general eye consultation, cataract screening & surgery, and laser procedures.
- Schedule and Locations: Clearly display the clinic schedule and locations for Premiere Eye Referral Center, Ideal Vision Center, and Maxicare Primary Care Clinics.

## Style Guidelines:

- Primary color: Soft blue (#89BBE3) to evoke calmness and trust.
- Background color: Off-white (#F0F4F8) for a clean and professional feel.
- Accent color: A slightly darker, analogous blue (#60A6D9) to highlight important information.
- Body font: 'PT Sans' (sans-serif) for clear and readable body text.
- Headline font: 'Belleza' (sans-serif) for section titles, giving an elegant feel; use 'Alegreya' for longer text
- Use professional and clear icons related to eye care and medical services.
- Apply subtle fade-in animations when sections scroll into view to create a smooth user experience.
